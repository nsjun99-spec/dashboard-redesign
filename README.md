# dashboard-redesign

Finance Report Automation Preview

## 설치 및 실행

1. 프로젝트 루트로 이동

```bash
cd /workspaces/dashboard-redesign
```

2. 의존성 설치

```bash
npm install
```

3. 개발 서버 실행

```bash
npm run dev
```

4. 브라우저에서 열기

`http://localhost:5173`

## 배포

- Vercel이나 Netlify에 연결하면 자동으로 새 URL이 생성됩니다.
- 현재 코드베이스는 React + Vite + Tailwind 기반으로 구성되어 있습니다.
